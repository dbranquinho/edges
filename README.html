Fist Challenge

In this challenge, suppose we are looking to do social network analysis for prospective customers. We want to extract from their social network a metric called closeness centrality.

Centrality metrics try to approximate a measure of influence of an individual within a social network. The distance between any two vertices is their shortest path. The farness of a given vertex v is the sum of all distances from each vertex to v. Finally, the closeness of a vertex v is the inverse of the farness.

The first part of the challenge is to rank the vertices in a given graph by their closeness. The graph is provided in the attached file; each line of the file consists of two vertex names separated by a single space, representing an edge between those two nodes.

Density

The density of a graph is the number of existing edges divided by the number of possible ones (assuming no duplicates or loops)

Connectedness takes one or more graphs and returns the Krackhardt connectedness scores

Centrality

Centralization returns the centralization GLI (graph-level index) for a given graph in dat, given a (node) centrality measure FUN. Centralization follows Freemans (1979) generalized definition of network centralization, and can be used with any properly defined centrality measure.

centralization(net,degree,mode="graph")
## [1] 0.1612903

Closeness Centrality

Closeness Centrality (CLC) is a category of measures that rate the centrality of a node by its closeness (distance) to other nodes. Closeness Centrality decreases if either the number of nodes reachable from the node in question decreases, or the distances between the nodes increases.
